Bootstrap 5 with Sass Setup

  1. Run npm install bootstrap sass in your project folder.
  2. Create a scss folder and add a custom.scss file.
  3. Import Bootstrap in custom.scss: 
    @import "../node_modules/bootstrap/scss/bootstrap";
  4. Compile your SCSS to CSS using sass scss/custom.scss css/custom.css.
  5. Link css/custom.css in your HTML.
